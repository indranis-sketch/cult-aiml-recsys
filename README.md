# Cult Open Projects 2026 - AI/ML

**Topic:** Recommendation Systems for Personalized Content Discovery

## Project Structure

*   `dataset/`: Netflix Prize Dataset
*   `notebooks/`: Jupyter notebooks for exploratory data analysis (EDA) and experimental model building.
*   `src/`: Main source code containing data pipelines, models, evaluation metrics, and the recommendation generator.
*   `docs/`: Drafts for technical reports and presentations.

## Getting Started

1.  Clone this repository or navigate to this directory.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Place your Netflix Prize Dataset files in the `dataset/` directory.
4.  Launch Jupyter Notebook to start exploring `notebooks/01_eda.ipynb`.
